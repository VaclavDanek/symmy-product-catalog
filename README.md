# Info
Simple product catalog based on React

## Main features
* React
* TypeScript
* Vite
* Eslint
* Ant Design + icons
* i18next
* Tailwind
* TanStack Query
* react-router-dom
* react-lazy-load-image-component

## Installation and startup
1. npm install
2. npm run dev
3. visit [http://localhost:5173/](http://localhost:5173/)

## Build
1. npm run build
2. go to "dist" folder, located in the root directory
